# CLI Student Management System

A lightweight, command-line interface (CLI) application built in Python for managing student records. This project uses Python's native dictionary data structures to handle data storage in-memory, making it fast, simple, and dependency-free.

---

## 🚀 Features

* **Add New Students:** Input and store student details (ID, Name, Age, Grade/Course).
* **View Student Records:** Display details of a specific student or view the entire student directory.
* **Update Information:** Modify existing student records seamlessly.
* **Delete Records:** Remove student profiles from the system using their unique ID.
* **In-Memory Storage:** Powered entirely by Python dictionaries for efficient data handling.
